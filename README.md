# 🔐 CIPHER.GEN — Cyberpunk Password Generator

A futuristic and secure password generator website built using **HTML, CSS, and JavaScript** with a stunning cyberpunk-inspired UI, animated backgrounds, glowing effects, and advanced password generation features.

## 🚀 Features

* Generate strong and secure random passwords
* Adjustable password length slider
* Toggle options for:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Symbols
* Password strength indicator
* Entropy calculation system
* Copy password to clipboard
* Password generation history log
* Interactive ripple effects and glowing buttons
* Animated particle background using Canvas
* Glassmorphism cyberpunk UI
* Fully responsive design
